#### bootstrap 사용법

bootstrap을 구동하기 위한 뼈대 코드는 다음과 같다.

```html
<!DOCTYPE html>
<html>

  <head>
  	<meta charset="utf-8">	<!-- 한글을 위해 필요 -->
    <title>Bootstrap 101 Template</title>
    <link href="bootstrap.min.css" rel="stylesheet" media="screen">
  </head>

  <body>
    <h1>안녕, HTML!</h1>
  </body>
</html>
```
